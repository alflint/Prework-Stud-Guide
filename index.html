HTML

  • HTML stands for Hypertext Markup language.
  • HTML is one of the three cornerstone languages used in web development and is used to create the structure of a webpage.
  • An element is a type of content on a webpage. Elements include: text elements, image elements, header elements, list elements.
  • Common types of elements used to divide and describe a page include: meta-related elements, content division elements and semantic elements.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user t navigate the webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • DOCTYPE is not case-sensitive because it is a declaration. Declarations are not case-sensitive; HTML elements are case-sensitive.
  • An HTML attribute defines additional information about an element and are always defined at the beginning of an element, never the closing element.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • Metadata is not typically seen by users but it contains important information about accessibility, search engines, and performance.
  • The title element defines the title for the webpage, is required, and is extremely important for search engine optimization (SEO).
  • Search engine optimization (SEO) is the process of improving the ranking of a website in search engines such as Google or Bing. Better SEO results in your website results in your website more frequently showing up in searches.
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, description, keywords, author and viewpoint.
  • The script element embeds client-side scripts in the HTML document.
  • To create a new branch: git status, git checkout main, git checkout -b feature/add-css.

CSS

  • CSS stands for Cascading Style Sheets.
  • Cascading Style Sheets (CSS) is a program used by developers to define how webpage content is presented to users.
  • CSS defines colors, aesthetic, visual layout and organization
  • We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • CSS is a language that defines the design and layout of webpages.
  • CSS allows us to style page elements how we want, including layout, color and style.
  • There are three ways to style a webpage using CSS:inline CSS, internal CSS style sheet, and external CSS Style Sheet.
  • Inline CSS - inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. It is typically used when a developer wants to style an element directly in the HTML file.
  • Inline is useful for quick and permanent changes but are less flexible than external and internal style sheets because each inline style must be edited individually whenever you make a design change.
  • Internal Style Sheet - contains CSS rules for the webpage in the head section of the HTML file and allows for the possibility to embed an entire stylesheet directly in the HTML file.
  • Internal Style Sheet rules only apply to that page. You can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
  • A single change to the CSS rule will apply to all tagged page elements using internal.
  • External CSS Style Sheet - most common way to style a webpage and is preferred because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • Separate of concerns - refers to separating code into different files based on use. Each section of code should have its own responsibility.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations - contain two components: the CSS property we want to apply and the value of the property.
  • The margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside of an element and is not the same thing as the margin property.
  • Class attributes allow us to share a CSS rule to any elemen we choose by assigning the rule to a class attribute with a class selector. Then the class attribute can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites. Using JS, you can power your websites with effects, interactivity, and data communication.
  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive.
  • JavaScript is a programming language that allows us to implement more complex features on web pages
  • JavaScript allows you to control multimedia, animate images, and dynamically control content.
  • Variable - a named container that allows us to store data in our code. We use the var keyword to declare a variable.
  • All strings are surrounded by quotation marks.
  • The console.log method - used to output the value of topics to the web console.
  • Open Chrome DevTools by right clicking and selecting "Inspect" or press command+option+I.
  • Control flow is the order in which a computer executes code in a script.
  • In JS, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • Conditional Statements and loops - controls the order in which code is executed, or if it is even executed at all.
  • An if statement is a way of introducing decision-making into our code. The code will only execute if the statement meets a condition or is true.
  • if a condition isi truthy, the computer will do whatever the action is that we ask between the curly brackets. It not, it will ignore it.
  • Array - a single variable that is used to hold a group of data.
  • Index - the location of the data item in the array.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • The conditional if statement is used to interrupt the control flow by executing a block of code a specific number of times.
  • The first statement determines the starting point for the loop.
  • The second statement is the condition.
  • The final statement allows the array to iterate over each time.
  • Commenting out code makes it inactive without deleting it using /* before and */ after the block of code.
  • Function - a set of instructions that tells the computer how to perform a certain task.
  • Two requirements for a Javascript function: 1. Define the function and 2. Call the function.
  • When we declare a function in our JavaScript code, we are creating a reusable block of code that performs a specific task. However, declaring a function does not mean that the code with automatically execute, we need to call the function.
